# Credit Card Fraud Detection using Machine Learning

## Business Problem
Credit card fraud leads to billions in financial losses every year. Financial institutions require intelligent systems capable of identifying fraudulent transactions quickly and accurately.

## Objective
Develop a machine learning model that detects fraudulent transactions while minimizing false positives.

## Tools & Technologies
- Python
- Pandas
- Scikit-Learn
- SMOTE (Handling Class Imbalance)
- Matplotlib & Seaborn

## Key Skills Demonstrated
✔ Data Cleaning  
✔ Feature Engineering  
✔ Handling Imbalanced Data  
✔ Machine Learning  
✔ Model Evaluation  
✔ Risk Analytics  

## Model Performance
- F1 Score: **0.88**
- AUC Score: **0.98**
- High fraud detection recall with minimal missed fraud cases.

## Business Impact
This model can help financial institutions:

- Detect high-risk transactions
- Reduce financial losses
- Improve customer trust
- Strengthen fraud prevention strategies

Public dataset from ULB Machine Learning Repository:
